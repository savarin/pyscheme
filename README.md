# pyscheme

Minimal implementation of Scheme for Fibonacci generation.


## Quickstart

Set up with `uv`.

```python
uv venv --python 3.12
source .venv/bin/activate
uv pip install ".[dev]"
```

Run tests.

```bash
./bin/test.sh
```
